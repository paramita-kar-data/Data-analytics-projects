# Python-Customer-Segmentation-Analysis/README.md

This project analyzes customer purchase behavior to identify high-value, medium-value, and low-value customers.

## Objectives
- Identify top customers based on revenue
- Segment customers into value groups
- Analyze revenue contribution by segment

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
High-value customers contribute the majority of total revenue, highlighting the importance of customer retention strategies.
